04 · How the Product is Composed
What Terra Ferra builds, what it integrates, what it subscribes to. Reference diagram for engineering scope, fundraising conversations and CTO alignment.

How to read this

Terra Ferra is composed of three kinds of building blocks. TF Core is proprietary — the IP Terra Ferra owns and the work the founding engineering team builds in-house (pre-order engine, co-creation, ambassador/commission logic, catalog, user data). Integrate covers third-party APIs and open-source models TF embeds in its own product to deliver features faster than building from scratch (visual search, payments, tax, fulfilment, sizing). Subscribe covers SaaS tools the TF team uses operationally (creator discovery, ambassador management, analytics). Infrastructure is the foundational layer everything runs on (cloud, database, search, auth).

Every block named below is either already verified in Section 05 — AI Discovery Tools or is a category-standard service. Specific vendor choices within categories (Stripe vs Adyen, AWS vs GCP, Algolia vs FashionCLIP) are CTO-level decisions, not founder-level.

Open vendor decisions — CTO calls

AI Discovery

Syte (managed) vs FashionCLIP (self-hosted)

Both deliver visual similarity for fashion. Syte is faster to integrate and vendor-supported (Farfetch, SHEIN, PrettyLittleThing use it). FashionCLIP is open-source, MIT-licensed — TF owns the IP, no recurring vendor cost. Trade-off: speed vs ownership.

Payments

Stripe vs Adyen vs Braintree

Stripe is the default for ecommerce in 2026 and supports authorise-now-capture-later (required for pre-orders). Adyen is the alternative for high-volume EU operations. Braintree (PayPal) trails on developer experience.

Cloud + Database

AWS vs GCP vs Vercel + Neon

AWS gives the broadest service catalog and is the default for fashion DTC. Vercel + Neon (Postgres) is the modern serverless stack — faster to ship, scales to mid-sized traffic. CTO call based on team familiarity.

Search & Recommendations

Algolia vs Constructor vs in-house

Algolia is the standard for fashion DTC site search (END., Gymshark). Constructor is the AI-commerce alternative with stronger personalisation (Sephora, Gap, Rothy's). In-house only if TF has compelling differentiation reason.

Fulfilment

ShipBob vs ShipStation vs designer-fulfilled

ShipBob (3PL) takes inventory burden off designers. ShipStation is a multi-carrier label layer for designer-fulfilled orders. Pre-order model + small designers → designer-fulfilled is the default; 3PL becomes relevant when GMV justifies it.

Mobile stack

Native (Swift + Kotlin) vs React Native vs Flutter

Native gives best performance and platform-API access (camera for visual search, push notifications). React Native and Flutter let one team ship both platforms faster. CTO call based on hiring profile.
